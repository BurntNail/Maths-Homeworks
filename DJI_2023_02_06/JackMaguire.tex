\qs{
To get equal balls, one of the ball counts must be 1 away from the other 2 ball counts, with sign changes, which leaves 2 possibilities:
\begin{multicols}{2}
	\noindent
	\[ n = 5, A_n = 12 \]
	\begin{itemize}
		\ii Green A->B
		\ii Yellow B->A
	\end{itemize}
	The chance is:
	\[ = \frac{5}{12} * \frac{3}{10} = \frac{1}{8} \]
	
	\columnbreak
	\[ n = 2, A_n = 9 \]
	\begin{itemize}
		\ii Red A->B
		\ii Green B->A
	\end{itemize}
	The chance is:
	\[ = \frac{4}{9} * \frac{1}{10} = \frac{4}{90} \]
\end{multicols}
Therefore, the total chance is:
\[ = \frac{1}{8} + \frac{4}{90} = 0.1694 = 16.9\% \]
}
